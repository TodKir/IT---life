Вторая и последующие сессии


Вторая и последующие сессии строятся по следующему плану:

  • Краткая оценка текущего состояния клиента
  • Связь текущей сессии с предыдущей
  • Установление повестки дня
  • Анализ домашнего задания
  • Обсуждение вопросов повестки дня, определение домашнего задания и периодическое подведение итогов
  • Заключительное подведение итогов и запрос у клиента для обратной связи

Краткая оценка текущего состояния


Оценка состояния обычно проходит быстро, ее можно сочетать с кратким обзором прошедшей недели. Сравниваем то, что говорит клиент с результатами тестов, опросников.

Важно сравнить результаты предыдущего исследования с текущими результатами.

Если появляется какой-то новый запрос от клиента – включаем его в повестку дня.


Установление повестки дня

Важный вопрос каждой сессии – какую проблему клиента рассмотреть. Не смотря на то, что важно привлекать клиента к определению проблемы для обсуждения и цели, необходимо избегать обсуждения проблем, которые:

  • ухудшают состояние клиента
  • решение которых не способствует значительному улучшению состояния во время сессии
  • пациент может решить самостоятельно
